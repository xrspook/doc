811上海地铁交通路线贴士

作者: xrspook

日期: 2012-08-02 20:34:00+08:00

分类: 点点图片

标签: alberto_del_rio, wwe, 2012, smackdown, house_show, shanghai, china, 交通, 地铁, 上海, 中国, 巡演, 811

如果你是高富帅，有大把大把的money，习惯打的解决问题的，你可以完全忽略下面的内容了。

下面的内容完全基于上海地铁、19977的根据地以及各个火车站/机场。

穷苦逼伤不起，从中国其它地方到上海已经让很多外地摔迷筋疲力尽，到了上海，肿么办呢？

今天我特意看了看地图，看了看地铁线，知道该怎么做了，特此做个小贴士让大家找得着北（我第一次去上海啊～～～～）。

地铁交通及票价

上海火车站到浦江之星，4元：地铁4号线（上海火车站-鲁班路），共14站

上海火车站 - 宝山路 - 海伦路 - 临平路 - 大连路 - 杨树浦路 - 浦东大道 - 世纪大道 - 浦电路 - 蓝村路 - 塘桥 - 南浦大桥 - 西藏南路 - 鲁班路

上海南站到浦江之星，4元：地铁1号线转地铁4号线（上海南站-上海体育馆[换乘]-鲁班路），共7站

上海南站 - 漕宝路 - 上海体育馆 - 上海体育场 - 东安路 - 大木桥路 - 鲁班路

上海西站到浦江之星，4元：地铁11号线转地铁4号线（上海西站-曹杨路[换乘]-鲁班路），共14站

上海西站 - 真如 - 枫桥路 - 曹杨路 - 金沙江路 - 中山公园 - 延安西路 - 虹桥路 - 宜山路 - 上海体育馆 - 上海体育场 - 东安路 - 大木桥路 - 鲁班路

虹桥机场1号航站楼到浦江之星，4元：地铁10号线主线转地铁4号线（虹桥1号航站楼-虹桥路[换乘]-鲁班路），共13站 虹桥1号航站楼 - 上海动物园 - 龙溪路 - 水城路 - 伊犁路 - 宋园路 - 虹桥路 - 宜山路 - 上海体育馆 - 上海体育场 - 东安路 - 大木桥路 - 鲁班路

浦东机场到浦江之星，7元：地铁2号线转地铁4号线（浦东国际机场-世纪大道[换乘]-鲁班路），共21站

浦东国际机场 - 海天三路 - 远东大道 - 凌空路 - 川沙 - 华夏东路 - 创新中路 - 唐镇 - 广兰路 - 金科路 - 张江高科 - 龙阳路 - 世纪公园 - 上海科技馆 - 世纪大道 - 浦电路 - 蓝村路 - 塘桥 - 南浦大桥 - 西藏南路 - 鲁班路

浦江之星到东方体育中心，4元：地铁4号线转地铁8号线（鲁班路-西藏南路站[换乘]-东方体育中心），共6站

鲁班路 - 西藏南路 - 耀华路 - 成山路 - 杨思 - 东方体育中心

东方体育中心到旭日屋，4元：地铁8号线转地铁9号线（东方体育中心-陆家浜路[换乘]-嘉善路），共9站

东方体育中心 - 杨思 - 成山路 - 耀华路 - 西藏南路 - 陆家浜路 - 马当路 - 打浦桥 - 嘉善路

在旭日屋聚餐后回浦江之星的话，时间关系，估计是打车了，路很顺，也就是：襄阳南路-肇嘉浜路-打浦路，从地图上看的话，大概就2公里路程（非直线连接哦！），如果是一帮人，散步也未尝不可，我觉得。
